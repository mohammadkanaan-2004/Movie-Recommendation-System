# 🎬 Movie Recommendation System

## 📌 Overview
This project is a context-based movie recommendation system that suggests movies based on user-generated tags using the MovieLens dataset.

It uses Natural Language Processing (NLP) techniques to understand movie context such as:
- Mood
- Genre
- Themes

---

## 🚀 Features
- Context-aware recommendations
- TF-IDF vectorization
- Cosine similarity matching
- Tag-based filtering

---

## 🧠 Tech Stack
- Python
- Jupyter Notebook
- Pandas, NumPy
- Scikit-learn
- Matplotlib
